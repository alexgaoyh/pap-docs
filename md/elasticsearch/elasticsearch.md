Spring Boot 自定义 Starter 组件 [pap-es-spring-boot-starter](https://gitee.com/alexgaoyh/pap-es-spring-boot-starter) , 包含如下功能：

- 环境配置
- IK 分词器
- Ingest Attachment Processor Plugin
- Synonym 同义词
- @Mapping @Setting
- 分组
- constant_score
- DIS_MAX
- copy_to
- match_phrase.slop
- function_score
- boosting
- 锁
- Nested Object
- 父子文档
- completion suggest
- 实际场景-具体应用
- bulk批处理
